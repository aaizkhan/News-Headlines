## About The Project
Flutter MVVM project for News Headlines. REST Api for News is used in this project.


### App Demo

### Api for News

1. Get free api from here :  [https://newsapi.org/](https://newsapi.org/)
2. Clone the repo
   ```sh
   git clone https://github.com/aaizkhan/News-Headlines.git
   ```
